# Remote Config manifest templates

Use SHA256 of the **gunzipped .sqlite** files, not the .gz files. Replace URL placeholders after upload.

## dictionary_manifest_tr
```json
{
  "language": "tr",
  "version": 2,
  "url": "https://emreyilmaz1912.github.io/ucurtma-legal-pages/dictionaries/turkish_words_v1.sqlite.gz",
  "checksum": "sha256:05cf59188d2a8fc9a8c6456624d20ff377264431e5b10e84d2bd63a557ee2501",
  "enabled": true,
  "format": "sqlite"
}
```
